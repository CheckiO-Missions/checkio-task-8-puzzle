'8-puzzle' é um quebra-cabeça deslizante que consiste de uma armação com 'peças numeradas dispostas aleatoriamente, com uma peça faltando. O 'objetivo do quebra-cabeça é colocar as peças na ordem correta (ver 'imagem), deslizando as peças para utilizar o espaço vazio. Você pode ler mais sobre esse tipo de quebra-cabeça na Wikipedia

Nosso quebra-cabeça consiste de uma matriz 3x3 com números de 1 a 8. Zero é a célula vazia. Você pode "mover" a célula vazia em quatro direções: cima--"U", baixo--"D", esquerda--"L", e direita--"R". Você precisa retornar a sequência de movimentos que resolve o quebra-cabeça. A solução deve ser dada como uma sequência(string) de símbolos ("UDLR") que descreve seus movimentos.

Entrada: Uma matrix com números de 1 a 8 inteiros (int), como uma lista de listas.

Saída: O percurso da célula vazia como uma palavra(string).

Exemplo: 

checkio([[1, 2, 3],
         [4, 6, 8],
         [7, 5, 0]]) == "ULDR"

Como é usado: O uso mais obvio dos conceitos desse problema esta na criação de um robô para resolver esse tipo de quebra-cabeça; entretanto, esse problema é uma maneira divertida de aprender algo novo, já que esse tipo de quebra-cabeça é um problema clássico de modelagem de algoritmos que envolvem heurísticas.

Precondição:
len(puzzle) == 3
all(len(row) == 3 for row in puzzle)
