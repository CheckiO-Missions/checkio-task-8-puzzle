Восемьнашки (от слова Пятнашки) - Представляет собой набор одинаковых квадратных костяшек с нанесёнными числами, заключённых в квадратную коробку. Длина стороны коробки в три раза больше длины стороны костяшек для набора из 8 элементов, соответственно в коробке остаётся незаполненным одно квадратное поле. Цель игры — перемещая костяшки по коробке, добиться упорядочивания их по номерам, желательно сделав как можно меньше перемещений.

Головоломка представлена, как матрица 3 на 3 с цифрами от 1 до 8. 0 - это пустое поле. Вы можете "двигать" пустое поле в четырех направлениях: вверх--"U", вниз--"D", влево--"L", и вправо--"R". Вам нужно найти последовательность ходов, чтобы решить головоломку. Решение должно быть представлено, как строка букв ("UDLR") описывающих действия.

Входные данные: Матриц с цифрами от 1 до 8, как список (list) списков с целыми числами.

Выходные данные: Последовательность действий, как строка.

Пример: 

checkio([[1, 2, 3],
         [4, 6, 8],
         [7, 5, 0]]) == "ULDR"

Как это используется: Самое очевидное использование - это создание бота для решения подобных головоломок. Однако это также моделирование с эвристикой (если вы используете ее конечно).

Предусловия:
len(puzzle) == 3
all(len(row) == 3 for row in puzzle)
